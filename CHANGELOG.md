# ListFlow Changelog

## v1.1.0
- Fixed app icon now displays correctly in the taskbar and window title bar
- Fixed notes not loading correctly after restarting the app
- Fixed PDF export not showing notes on resolved tasks
- Added completion notes — prompted when marking a task as solved
- Added image attachments to tasks
- Added move task between lists option in the task menu
- Added JSON import/export for sharing lists with others
- Added version selector in Settings to install any previous version
- Added "Check for updates" button in Settings
- Removed Hidden (Later) feature
- Improved export dropdown now shows list names correctly

## v1.0.0 — Initial Release
- Create and manage multiple task lists
- Tasks with priority levels (Low / Medium / High), due dates, and follow-up reminders
- Completion notes when marking tasks as solved
- Notes and image attachments on tasks
- Edit note timestamps
- Today view showing all tasks due or overdue across all lists
- Move tasks between lists
- PDF export for bug tracking reports
- Plain text export for sharing
- JSON import/export to share lists with teammates
- Desktop notifications for follow-up and due dates
- Daily morning digest notification
- Auto-update support — checks for updates on startup
- Version selector to install any previous version
