# ListFlow Changelog

## v1.0.0 - Initial Release
- Create and manage multiple task lists
- Task details: priority, due dates, follow-up reminders, notes, images
- Completion notes when marking tasks as solved
- Today view showing all tasks due or overdue
- PDF and text export for bug tracking
- JSON import/export to share lists with others
- Move tasks between lists
- Desktop notifications for follow-up and due dates
- Daily morning digest notification
- Auto-update support
