# ListFlow

A professional task and list management app for Windows.

## Features

- Multiple lists (Work, Personal, Projects, etc.)
- Tasks with priority levels, due dates, follow-up reminders
- Completion notes for bug tracking workflows
- Notes and image attachments on tasks
- PDF export for bug reports
- JSON import/export to share lists with teammates
- Move tasks between lists
- Desktop notifications
- Auto-updates

## Installation

1. Download the latest `ListFlow-v{VERSION}.zip` from [Releases](../../releases/latest)
2. Extract the ZIP anywhere on your PC
3. Run `ListFlowWPF.exe`
4. No installer needed — just run and go

## Requirements

- Windows 10 or later
- .NET 10 Runtime ([download here](https://dotnet.microsoft.com/download/dotnet/10.0))

## Updating

The app checks for updates automatically on startup. When an update is available it will ask if you want to install it. Your lists and tasks are never touched during updates.
